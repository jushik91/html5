1. display 속성

none 보이지않게 줄바꿈
block 줄바꿈  
inline 한줄 margin(left, right), padding(right, bottom, left)
inline-block 한줄(안에는 여러줄) width, height, margin(top, right, bottom, left)

 

2. 태그로 나눈 display 속성

block 형식 div, h1~h6, p, 목록, 테이블, 폼
inline 형식 span, a, input, 글자형식
inline-block형식 이미지태그, 멀티미디어태그
